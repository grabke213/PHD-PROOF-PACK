# PHD Precision Certificate v1 (Offline-first)

## Run locally
Open `index.html` in a local web server (recommended).
- If you have Python:
  - `python -m http.server 8080`
  - Visit: http://localhost:8080/phd_precision_certificate_v1/

## Deploy to GitHub Pages
1) Create a repo (or use existing).
2) Upload contents of this folder to the repo root.
3) Settings → Pages → Deploy from branch → `main` / `/ (root)`.
4) Open the Pages URL on your phone.

## Offline
- First open the app once while you have internet (so it caches).
- After that, it will work offline: jobs + photos + signatures.

## PDF Proof Pack
Click **Generate PDF (Print)** → choose **Save as PDF**.
